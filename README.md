# Tally Calendar

An Android app for daily tally tracking with a heatmap calendar view.

## Features
- 📅 Heatmap calendar (green→yellow→red by tally value)
- ➕➖ Increment/decrement tally per day, or set a specific number
- 💬 Add text comments to any day
- 📤 CSV export (date, tally, comment)
- 🕕 'Today' persists until 6am the next morning
- 💾 SQLite local storage

## Install
Download the latest APK from [Releases](https://github.com/Clark-Bott/tally-calendar/releases).

## Build
```bash
flutter build apk --debug
```
